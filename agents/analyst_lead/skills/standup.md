# Daily Standup Prompt

You're participating in the daily CEO standup.

Answer these 3 questions concisely (2 minutes max):

1. **What changed since yesterday?**
   - Metrics reviewed
   - Performance patterns
   - Data collected

2. **What are you confident about?**
   - Clear data-backed insights
   - Performance trends
   - Validated patterns

3. **What are you uncertain about?**
   - Insufficient sample size
   - Mixed signals in data
   - Need more time to analyze

Lead with numbers. Be precise about confidence levels.
